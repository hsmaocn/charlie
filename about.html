关于我们

我们是一个热爱周深音乐的歌迷组织，致力于收集和分享周深在各大平台上的精彩视频和资讯。 我们的使命是让更多人了解周深的音乐才华，并为歌迷提供一个交流和分享的平台。

如果你对我们的工作感兴趣，欢迎加入我们！

如何贡献内容？

我们非常欢迎各位歌迷朋友通过以下方式为这个项目贡献力量：

  1. Fork 我们的 GitHub 仓库：点击这里访问仓库。
  2. 在本地修改或添加内容（例如新增视频链接、描述等）。
  3. 提交 Pull Request，我们会审核并整合到主分支。

通过这种方式，你可以帮助完善这个数据库，让更多人发现周深的精彩内容！

贡献者鸣谢

每位贡献者的信息将会被记录在对应的视频卡片中。如果你希望留下自己的信息，请在提交时注明以下内容之一：

  • 你的 GitHub 用户名
  • 你的哔哩哔哩账户链接
  • 其他社交媒体账户（如微博、YouTube 等）

示例：视频由 @YourBilibiliName 提供。
